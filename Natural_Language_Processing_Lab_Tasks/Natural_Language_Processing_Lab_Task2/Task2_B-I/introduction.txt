CRF++
Linux版本的安装方法是：
                         i.              解压到某目录下
                       ii.              打开控制台，将当前目录切换到解压目录
                      iii.              依次输入命令：
./configure
make
su
make install
                            注：需要root权限才能成功安装。


由于这里使用 python 工具包进行训练和测试，需要安装 python 工具包。进入 python 文件夹，运行以下命令安装：
python setup.py build
sudo python setup.py install
